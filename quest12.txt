O comando aloha[2] = value; é válido, pois o vetor aloha[0] deslocado duas casas vai receber o valor fornecido por value. O scanf("%f", &aloha) é válido, pois ele vai receber um valor para e armazenar no primeiro elemento do vetor. O aloha = value é inválido, pois estaria atribuindo um valor a um endereço.  O printf("%f", aloha) é válido, pois vai mostrar o endereço do vetor. O coisas[4][4] = aloha[3] é válido, pois vai atribuir o valor do elemento encontrado no terceiro espaço do vetor aloha à matriz coisas no espaço da quarta linha e quarta coluna. coisas[5] = aloha; não é válido, pois não foi especificado o elemento da coluna da matriz. O pf = value não é válido, pois o value deveria ter um & no início. pf = aloha é válido, pois o ponteiro vai apontar para o primeiro elemento do vetor aloha.
